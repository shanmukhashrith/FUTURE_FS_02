# 🚀 Modern Client Lead Management System (CRM)

A full-stack CRM built during my development internship. 
Features a React frontend, Node.js backend, and MongoDB cloud storage.

## ✨ Features
- **Real-time Dashboard:** View lead analytics and conversion rates.
- **Lead Management:** Add new leads via a modern modal form.
- **Dynamic Status Updates:** Update lead status with instant UI feedback.
- **Dark Mode UI:** Professional SaaS design using Tailwind CSS.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Lucide Icons.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (Cloud).

## 🏃‍♂️ How to Run Locally
1. Clone the repo.
2. In `/backend`, run `node server.js`.
3. In `/frontend`, run `npm run dev`.

## Setup & Installation
1. Clone the repository: `git clone https://github.com/shanmukhashrith/FUTURE_FS_02.git`
2. Navigate to the backend folder and install dependencies: `cd backend && npm install`
3. Start the backend server: `node server.js`
4. In a new terminal, navigate to the frontend folder: `cd frontend && npm install`
5. Start the Vite React app: `npm run dev`
